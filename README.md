# Warranty & Bills Tracker

A full-stack Warranty and Bills Tracker project built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## Features

- User Signup & Login
- Add Products
- Edit Products
- Delete Products
- Search Products
- Product Categories
- Warranty Expiry Tracking
- Expiring Soon Alerts
- Dashboard Insights
- Dynamic Product Cards

## Backend Features

- REST API Integration
- CRUD Operations
- Product Management
- User Authentication
- Dynamic Data Handling

## Project Purpose

This project helps users manage:
- warranties
- bills
- expiry dates
- product information

in one dashboard.
